开源 Agent 框架 OpenClaw 正式发布 1.0 版本。这是首个原生支持 Windows/macOS/Linux 三平台的 Agent 框架。

新版本引入了 Pipeline 2.0 引擎，支持更复杂的多工具编排。目前已集成 Codex API、Claude API、OpenAI API 等主流模型接口。
