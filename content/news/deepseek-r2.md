DeepSeek R2 在 AIME、GPQA、SWE-bench Verified 等基准上达到或超过 GPT-5、Claude 4.5 水平。模型权重与推理代码全部采用 MIT 协议开源。

API 价格为同类闭源模型的 1/30 左右，每百万 token 输入仅 0.14 元。这一价格预计将再次引发大模型 API 降价潮。
