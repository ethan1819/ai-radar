Anysphere 旗下 AI 代码编辑器 Cursor 今日宣布，付费开发者用户突破 100 万，年度经常性收入（ARR）突破 5 亿美元。

Cursor 团队表示新一轮融资将用于扩展 Agent 能力，让 AI 自主完成从需求到 PR 的完整开发流程。其最新模型 Composer 2 在 SWE-bench 任务中已经能独立解决 60% 的真实 GitHub issue。
