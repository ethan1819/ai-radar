Anthropic 宣布 Claude 4.5 Sonnet 在 API 和 Claude.ai 全面开放。1M token 上下文进入 GA，可处理整本书籍或代码仓库级别的输入。

新模型在工具调用稳定性上提升了 40%，并引入了"思考块（thinking block）"机制让推理过程对开发者可见，便于调试和审计。

企业方面，Claude 4.5 接入 AWS Bedrock、Google Vertex AI 和 Microsoft Azure，三大云平台同步上线。
