## 什么是 Cline?

Cline 是 VS Code 上的自主 AI Agent 扩展。与传统代码补全不同,Cline 能自主理解任务、修改代码、执行命令、处理文件。

## 主要功能

Cline 通过 MCP 协议可以扩展各种工具能力。它支持自主完成从需求理解到代码提交的完整流程。

## 适合谁用?

需要 AI Agent 辅助编程的开发者,尤其是对 Agent 自主性有要求的用户。

## 总结

Cline 是当前 VS Code 生态中最成熟的 Agent 扩展。
