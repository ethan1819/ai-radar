## 什么是 OpenClaw?

OpenClaw 是首个原生跨平台的开源 Agent 框架,支持 Windows/macOS/Linux 三平台。最新 1.0 版本引入了 Pipeline 2.0 引擎。

## 主要功能

OpenClaw 的核心是模块化 Pipeline 设计,你可以像搭积木一样组合不同的工具和模型。

## 适合谁用?

想要构建自定义 Agent 工作流的开发者。

## 总结

OpenClaw 是 Agent 框架领域值得关注的新项目。
