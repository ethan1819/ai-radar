## 什么是 GitHub Copilot?

GitHub Copilot 是微软与 OpenAI 联合推出的 AI 编程助手,2022 年发布以来成为最广泛使用的 AI 编程工具之一。

## 主要功能

Copilot 的核心是代码补全——你在编辑器中写代码时它会预测你的意图给出建议。2026 年的版本已支持 Agent 模式,能自主完成复杂任务。

## 适合谁用?

所有写代码的开发者。如果你用 VS Code,Copilot 是最无缝的选择。

## 总结

Copilot 免费版已经能满足日常编码需求,Pro 版性价比很高。
