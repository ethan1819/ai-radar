## 什么是 AI Agent?

AI Agent 是能自主执行任务的 AI 系统。不同于被动回答问题的聊天机器人,Agent 可以主动规划、执行操作、使用工具。

## 2026 最值得关注的 Agent

**Codex Desktop** — 编程 Agent 标杆,支持 MCP 协议扩展。

**Claude Computer Use** — 桌面操作 Agent,能直接操作浏览器和应用。

**Cline** — VS Code 自主 Agent,开源活跃。

**OpenClaw** — 跨平台 Agent 框架。

**CrewAI** — 多 Agent 协作框架。

**AutoGPT** — 经典自主 Agent。

## 如何选择?

编程场景: Codex 或 Cline。桌面操作: Claude Computer Use。多 Agent 协作: CrewAI。
自定义框架: OpenClaw。
