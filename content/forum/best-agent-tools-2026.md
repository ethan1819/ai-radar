2026 年是 AI Agent 的爆发年，各大厂商和开源社区都推出了各具特色的 Agent 工具。

## 1. Codex Desktop
OpenAI 推出的编程 Agent，支持多文件自主修改和 MCP 协议扩展。

## 2. Claude Computer Use
Anthropic 的桌面操作 Agent，能直接操作浏览器和桌面应用。

## 3. OpenClaw
开源 Agent 框架，以模块化设计著称。

## 4. Cursor Composer
Cursor 内置的 Agent 模式，在代码编辑体验上最为流畅。
