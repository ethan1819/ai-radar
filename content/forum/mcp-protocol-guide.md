MCP（Model Context Protocol）是 Anthropic 提出的开放协议，让 AI 模型能够安全地访问外部工具和数据源。

## MCP 能做什么？
MCP 可以将文件系统、数据库、API 等外部资源桥接到 AI 模型中。Codex、Claude 等主流工具都已支持 MCP。

## 实际应用
通过 MCP，你可以让 AI 直接读写文件、查询数据库、调用外部 API。这比传统的 Function Calling 更标准化。
