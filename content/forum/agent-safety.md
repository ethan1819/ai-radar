AI Agent 让我们可以放心地让 AI 操作电脑了吗？

## 风险点

1. Agent 可能执行未预期的操作
2. 敏感信息可能被泄露
3. 第三方插件可能存在安全漏洞

## 最佳实践

- 在沙箱或隔离环境中运行
- 限制 Agent 的文件系统访问范围
- 审计 Agent 的操作日志
- 使用最小权限原则
