最近一直在用 Codex Desktop 和 Cursor 做 AI 编程，聊聊我的真实体会。

## Codex 的优势
Codex 最大的亮点是 Agent 模式——它能自主理解你的项目结构，在多文件间协调修改。对于复杂重构，Codex 比 Cursor 更可靠。

## Cursor 的优势
Cursor 在 Tab 补全的流畅度上无人能及，日常编码几乎不需要主动触发。它更像一个「超级智能的代码补全工具」，而 Codex 更像一个「自主编程的Agent」。
