对比 Cline、OpenClaw 和 CrewAI 三个框架。

**Cline** 是 VS Code 扩展,侧重编程场景,上手快,MCP 生态好。

**OpenClaw** 是通用跨平台 Agent 框架,Pipeline 引擎灵活。

**CrewAI** 主打多 Agent 协作,角色扮演式设计有创意。
